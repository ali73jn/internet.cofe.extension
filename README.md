# ✨ امکانات

## 📚 مدیریت حرفه‌ای بوکمارک‌ها

✅ نمایش بوکمارک‌ها به صورت کارت‌های زیبا و منظم

✅ نمایش پوشه‌ها و بوکمارک‌ها به دو حالت:

- 🟦 گرید (مربعی)
- 📋 لیستی (سطری)

✅ 🔎 جستجوی سریع بین تمام بوکمارک‌ها

✅ 📦 بزرگ و کوچک کردن کارت‌ها

✅ ↕️ جابه‌جایی کارت‌ها

✅ 🎨 شخصی‌سازی کامل نحوه نمایش کارت‌ها

✅ 💾 ذخیره و بازیابی تنظیمات نمایش

# 🎨 شخصی‌سازی محیط

با همیار کافینت می‌توانید ظاهر صفحه را کاملاً مطابق سلیقه خود تغییر دهید.

✨ امکانات:

- 🖼️ انتخاب تصویر زمینه دلخواه
- 💾 ذخیره و بازیابی تنظیمات
- 🕒 نمایش یا مخفی کردن کارت ساعت
- 🌤️ نمایش یا مخفی کردن کارت آب‌وهوا
- 🎯 شخصی‌سازی کامل محیط افزونه

---

# 🛠 ابزارهای داخلی

تعدادی از ابزارهای پرکاربرد به صورت کاملاً آفلاین داخل افزونه قرار گرفته‌اند.

## 📷 ابزارهای آفلاین

- 🗜️ کاهش حجم تصاویر
- 📐 کاهش رزولوشن تصاویر
- ✂️ برش تصاویر کج (Perspective Crop)

همچنین دسترسی سریع به ابزارهای آنلاین کاربردی نیز داخل افزونه قرار داده شده است.

---

# 📂 شخصی‌سازی کامل بوکمارک‌ها

تمام اطلاعات افزونه از پوشه‌ای با نام

```
netcofe
```

در ریشه بوکمارک‌های مرورگر خوانده می‌شود.

ساختار آن به شکل زیر است:

```
Bookmarks
└── netcofe
    ├── خدمات دولت
    │   ├── ...
    │   └── ...
    │
    ├── ثبت نام
    │   ├── ...
    │   └── ...
    │
    └── ...
```

## 💡 نحوه عملکرد

🟦 هر پوشه داخل **netcofe** یک کارت در صفحه اصلی ایجاد می‌کند.

🏷️ نام هر پوشه، عنوان همان کارت خواهد بود.

📑 بوکمارک‌های داخل هر پوشه، داخل همان کارت نمایش داده می‌شوند.

➕ با اضافه کردن پوشه جدید، کارت جدید ایجاد می‌شود.

➖ با حذف پوشه، کارت مربوطه حذف می‌شود.

✏️ با تغییر نام پوشه، عنوان کارت نیز تغییر خواهد کرد.

🎯 بنابراین بدون نیاز به هیچ تنظیمات اضافه، تنها با مرتب کردن پوشه **netcofe** می‌توانید ظاهر افزونه را دقیقاً مطابق سلیقه خود شخصی‌سازی کنید.

# 🚀 نصب افزونه

> ⚠️ **قبل از نصب، این بخش را با دقت مطالعه کنید.**

برای عملکرد صحیح تمامی امکانات افزونه، مخصوصاً ابزارهای آفلاین، **مسیر فایل‌ها نباید تغییر کند.**

---

## 📥 مرحله ۱ — دانلود و استخراج

پس از دانلود فایل افزونه:

✅ فایل ZIP را **مستقیماً داخل درایو C استخراج (Extract)** کنید.

مسیر صحیح باید دقیقاً به شکل زیر باشد:

```
C:\00-EXTENSION\01-bookmarks
```

❌ فایل را در مسیرهای زیر استخراج نکنید:

- Desktop
- Downloads
- Documents
- OneDrive
- داخل هر پوشه دیگری

در غیر این صورت برخی امکانات مانند:

- 🗜️ کاهش حجم تصاویر
- 📐 کاهش رزولوشن تصاویر
- ✂️ برش پرسپکتیو تصاویر

به درستی کار نخواهند کرد.

---

## 📂 مرحله ۲ — بررسی مسیر

پس از استخراج فایل، وارد درایو C شوید.

ساختار پوشه باید دقیقاً به شکل زیر باشد:

```
C:\
└── 00-EXTENSION
    └── 01-bookmarks
```

اگر این ساختار را مشاهده نمی‌کنید، فایل را در مسیر اشتباه استخراج کرده‌اید.

---

## 🌐 مرحله ۳ — باز کردن صفحه افزونه‌ها

مرورگر Chrome را باز کنید.

در نوار آدرس بنویسید:

```
chrome://extensions
```

و Enter را بزنید.

یا از منوی مرورگر وارد مسیر زیر شوید:

```
⋮
More Tools
    ↓
Extensions
```

---

## 🛠 مرحله ۴ — فعال کردن Developer Mode

در بالای سمت راست صفحه، گزینه

**Developer mode**

را پیدا کنید.

🔘 آن را روی **ON** قرار دهید.

پس از فعال شدن، سه دکمه جدید نمایش داده می‌شود:

📂 Load unpacked

📦 Pack extension

🔄 Update

---

## 📦 مرحله ۵ — نصب افزونه

روی دکمه

## 📂 Load unpacked

کلیک کنید.

پنجره انتخاب پوشه باز می‌شود.

به مسیر زیر بروید:

```
C:\00-EXTENSION\01-bookmarks
```

⚠️ بسیار مهم

✅ خود پوشه **01-bookmarks** را انتخاب کنید.

❌ وارد پوشه نشده و فایل‌های داخل آن را انتخاب نکنید.

پس از انتخاب پوشه، روی

**Select Folder**

کلیک کنید.

🎉 نصب افزونه به پایان رسید.

از این پس هر بار که یک تب جدید باز کنید، صفحه همیار کافینت نمایش داده خواهد شد.

---

# 🔄 بروزرسانی افزونه

قبل از نصب نسخه جدید، **حتماً از اطلاعات خود نسخه پشتیبان تهیه کنید.**

از داخل افزونه از دکمه‌های زیر استفاده کنید:

💾 خروجی تنظیمات

📑 خروجی بوکمارک

در صورت نیاز، پس از بروزرسانی می‌توانید تنظیمات و بوکمارک‌های خود را دوباره وارد (Import) کنید و همه چیز دقیقاً مانند قبل بازیابی خواهد شد.

---

# ❤️ هدف پروژه

همیار کافینت با هدف افزایش سرعت، نظم و دسترسی آسان‌تر به سایت‌ها، خدمات و ابزارهای موردنیاز کاربران کافینت طراحی شده است.

تمامی بخش‌های افزونه به گونه‌ای طراحی شده‌اند که بتوانید محیط کار خود را کاملاً مطابق سلیقه و نیاز خود شخصی‌سازی کنید و تنها با چند کلیک به تمامی ابزارها و سایت‌های موردنیاز دسترسی داشته باشید.
