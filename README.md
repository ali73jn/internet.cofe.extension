# همیار کافینت

همیار کافینت یک صفحه New Tab کاملاً شخصی‌سازی‌شده برای مرورگر Chrome است که با هدف افزایش سرعت انجام امور کافینت، مدیریت حرفه‌ای بوکمارک‌ها و دسترسی سریع به ابزارهای پرکاربرد طراحی شده است.

---

# امکانات

## مدیریت حرفه‌ای بوکمارک‌ها

- نمایش بوکمارک‌ها به صورت کارت‌بندی شده
- نمایش پوشه‌ها و بوکمارک‌ها به دو حالت:
  - گرید (مربعی)
  - لیستی (سطری)
- جستجوی سریع بین تمامی بوکمارک‌ها
- امکان بزرگ و کوچک کردن کارت‌ها
- جابه‌جایی کارت‌ها
- شخصی‌سازی کامل نحوه نمایش بوکمارک‌ها
- ذخیره تنظیمات نمایش و بازیابی آن‌ها

---

## شخصی‌سازی محیط

- انتخاب تصویر زمینه دلخواه
- ذخیره و بازیابی تنظیمات
- نمایش یا مخفی کردن کارت ساعت
- نمایش یا مخفی کردن کارت آب‌وهوا
- شخصی‌سازی کامل ظاهر صفحه

---

## ابزارهای داخلی

ابزارهای زیر بدون نیاز به اینترنت و به صورت آفلاین در افزونه قرار گرفته‌اند:

- کاهش حجم تصاویر
- کاهش رزولوشن تصاویر
- برش تصاویر کج (Perspective Crop)

همچنین دسترسی سریع به ابزارهای آنلاین پرکاربرد نیز در افزونه قرار داده شده است.

---

# شخصی‌سازی ساختار بوکمارک‌ها

تمامی بوکمارک‌های افزونه از پوشه‌ای با نام

```
netcofe
```

در ریشه بوکمارک‌های مرورگر خوانده می‌شوند.

ساختار به این صورت است:

```
Bookmarks
└── netcofe
    ├── خدمات دولت
    │   ├── ...
    │   └── ...
    │
    ├── ثبت نام
    │   ├── ...
    │   └── ...
    │
    └── ...
```

## نحوه عملکرد

- هر پوشه داخل **netcofe** یک کارت در صفحه اصلی ایجاد می‌کند.
- نام پوشه، عنوان کارت خواهد بود.
- بوکمارک‌های داخل هر پوشه، داخل همان کارت نمایش داده می‌شوند.
- با اضافه یا حذف پوشه‌ها، تعداد کارت‌ها تغییر می‌کند.
- با تغییر نام پوشه‌ها، عنوان کارت‌ها نیز تغییر خواهد کرد.

بنابراین می‌توانید تنها با مرتب کردن پوشه **netcofe**، ظاهر افزونه را کاملاً مطابق نیاز خود شخصی‌سازی کنید.

# نصب

## بسیار مهم

برای اینکه تمامی امکانات افزونه به درستی کار کنند، **فایل دانلود شده را حتماً مستقیماً در درایو C قرار دهید.**

مسیر صحیح باید به صورت زیر باشد:

```
C:\00-EXTENSION\01-bookmarks
```

فایل‌ها را در مسیر دیگری قرار ندهید، زیرا برخی از ابزارهای آفلاین افزونه در غیر این صورت به درستی اجرا نخواهند شد.

پس از دانلود:

1. فایل را از حالت فشرده خارج کنید.
2. پوشه **01-bookmarks** را داخل پوشه **00-EXTENSION** در درایو C قرار دهید تا مسیر زیر ایجاد شود:

```
C:\00-EXTENSION\01-bookmarks
```

3. وارد صفحه Extensions مرورگر شوید:

```
chrome://extensions
```

4. گزینه **Developer mode** را فعال کنید.
5. روی **Load unpacked** کلیک کنید.
6. پوشه زیر را انتخاب کنید:

```
C:\00-EXTENSION\01-bookmarks
```

# بروزرسانی

قبل از نصب نسخه جدید، حتماً از اطلاعات خود نسخه پشتیبان تهیه کنید.

از داخل افزونه، از گزینه‌های زیر استفاده کنید:

- خروجی تنظیمات
- خروجی بوکمارک

در صورت نیاز پس از بروزرسانی می‌توانید تنظیمات و بوکمارک‌های خود را دوباره بازیابی کنید.

---

# پشتیبان‌گیری

امکان تهیه نسخه پشتیبان و بازیابی موارد زیر وجود دارد:

- تنظیمات افزونه
- تصویر زمینه
- تنظیمات کارت‌ها
- تنظیمات نمایش بوکمارک‌ها
- اطلاعات بوکمارک‌ها

---

# هدف پروژه

هدف این پروژه ایجاد یک محیط سریع، مرتب و کاملاً شخصی‌سازی‌شده برای کاربران کافینت است تا تمامی سایت‌ها، ابزارها و خدمات موردنیاز تنها با چند کلیک در دسترس باشند.
